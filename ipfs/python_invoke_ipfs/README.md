
# python 调用ipfs

## 准备工作
1. 安装ipfs

[ipfs git地址](https://github.com/ipfs)

2. 安装Python插件
```
pip install ipfsapi

```

3. 启动ipfs节点
```
ipfs daemon
```

4. 运行test.py

5. 参考api

[py-ipfs-api github地址](https://github.com/ipfs/py-ipfs-api)
